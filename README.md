# BT-Controlled-Sumo-Bot 🤖🔥

A Bluetooth-controlled sumo robot designed for wireless navigation and robotic battle applications using Arduino and HC-05 Bluetooth communication.

---

## 🚀 Features

- Bluetooth-based wireless control
- Arduino-powered embedded system
- Real-time directional movement
- High-speed motor control
- Compact robotic chassis
- Mobile-controlled navigation

---

## 🛠️ Technologies Used

- Arduino UNO / Nano
- HC-05 Bluetooth Module
- L298N Motor Driver
- DC Motors
- Arduino IDE
- Embedded C

---

## ⚙️ Hardware Components

| Component     | Purpose                 |
| ------------- | ----------------------- |
| Arduino Board | Main controller         |
| HC-05 Module  | Bluetooth communication |
| L298N Driver  | Motor driver            |
| DC Motors     | Movement                |
| Battery Pack  | Power supply            |
| Robot Chassis | Structural body         |

---

## 🔌 Working

1. Mobile application sends commands through Bluetooth.
2. HC-05 receives the wireless signals.
3. Arduino processes the received commands.
4. Motor driver controls the motors accordingly.
5. Robot performs movement actions in real time.

## 🎮 Control Commands

| Command  | Action        |
| -------- | ------------- |
| Forward  | Move Forward  |
| Backward | Move Backward |
| Left     | Turn Left     |
| Right    | Turn Right    |
| Stop     | Stop Robot    |
